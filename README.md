# Code Fragrance - 代码香气分析器

一个创新的 VS Code 扩展，通过将代码特征转化为"香气特征"，帮助您更直观地理解和改进代码质量。

## 🌟 这个扩展是做什么的？

想象一下，如果代码也有香气：

- 优雅的代码像花香一样令人愉悦
- 稳定的结构如木质香般沉稳
- 创新的解决方案似果香般清新
- 高效的算法犹如东方调般深邃
- 灵活的接口设计如海洋调般自由

这个扩展就是通过这样的方式，帮助您：

1. 分析代码质量
2. 发现潜在问题
3. 保持团队一致的代码风格
4. 改进代码设计

## 📖 如何使用

### 1. 分析单个文件

1. 打开任意代码文件
2. 使用快捷键 `Cmd+Shift+F`（Mac）或 `Alt+Shift+F`（Windows/Linux）
3. 在右侧面板中查看分析结果：
   - 六种香气特征的评分
   - 具体问题的详细说明
   - 改进建议

### 2. 分析整个工作区

1. 使用快捷键 `Cmd+Shift+W`（Mac）或 `Alt+Shift+W`（Windows/Linux）
2. 查看整个项目的代码质量概览：
   - 各文件的评分分布
   - 整体代码风格统计
   - 潜在问题汇总

### 3. 配置团队风格

1. 点击左侧活动栏的烧杯图标
2. 选择"团队风格配置"
3. 可以选择以下方式之一：
   - 使用预设模板：
     - 严格模式：适合企业级项目
     - 平衡模式：适合一般项目
     - 灵活模式：适合创新项目
   - 使用配置向导：根据团队特点定制
   - 手动调整各项参数

### 4. 查看分析结果

分析结果包括：

1. **雷达图**：

   - 显示六个维度的得分
   - 可以直观看出代码的强项和弱项

2. **详细指标**：

   - 代码复杂度
   - 注释覆盖率
   - 函数长度
   - 命名规范性
   - 代码重复度
   - 设计模式使用情况

3. **改进建议**：
   - 具体问题定位
   - 改进方法建议
   - 最佳实践参考

## ✨ 特性

### 🌸 六种香调分析

- **清新调**：代码简洁性，包括行长度、缩进一致性和空白字符使用
- **木质调**：代码结构稳定性，评估复杂度和依赖关系
- **花香调**：设计模式优雅度，分析接口设计和封装性
- **果香调**：创新解决方案，评估算法创新性和独特性
- **东方调**：算法复杂度，分析逻辑密度和深度
- **海洋调**：接口灵活性，评估可扩展性和适应性

### 📊 详细指标

- 代码复杂度
- 注释比率
- 函数平均长度
- 命名一致性
- 代码重复度
- 设计模式使用度

### 🎯 主要功能

1. 实时代码分析
2. 可视化雷达图展示
3. 团队风格配置
4. 工作区分析支持

## 🚀 安装

1. 打开 VS Code
2. 按下 `Ctrl+P` / `Cmd+P`
3. 输入 `ext install code-fragrance`
4. 点击安装

## 📖 使用方法

### 分析当前文件

1. 打开任意代码文件
2. 点击活动栏中的烧杯图标
3. 查看代码香气分析结果

### 配置团队风格

1. 打开设置面板
2. 搜索 "Code Fragrance"
3. 根据团队需求调整配置项

### 快捷键

Windows/Linux:

- 分析当前文件：`Alt+Shift+F`
- 分析工作区：`Alt+Shift+W`

macOS:

- 分析当前文件：`Cmd+Shift+F`
- 分析工作区：`Cmd+Shift+W`

## ⚙️ 配置项

```json
{
  "codeFragrance.teamStyle": {
    "namingStyle": "camelCase",
    "maxFunctionLength": 30,
    "minCommentRatio": 0.1,
    "maxComplexity": 10
  }
}
```
